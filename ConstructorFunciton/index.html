Constructor Function

The constructor method is a special method of a class for creating and initializing an object instance of that class.

This Keyword

In JavaScript the "this" keypwrd refers to an object. Which object dependa on how "this" is being invoked(used or called). The "this" keyword refers to different objects depending on how it is used -

  • In an object method, "this" refers to the object.
  • Alone "this" refers to the global object.
  • In a function, in strict mode, this is undefined.
  • In an event, "this" refers to the element that received the event.
  • Methods like call(), apply() adn bind() can refer "this" to any object.
